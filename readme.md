## рҹҡҖ Usage

### Launching

```bash
bash run.sh
```

You'll be prompted with a legal notice. Accept to proceed.

### Module Walkthroughs

#### рҹҢҗ Web Application Scanner

1. Navigate to **WEB SCANNER** tab
2. Enter target URL (e.g., `https://yourtestsite.com`)
3. Click **в–¶ START SCAN**
4. Watch the live progress bar вҖ” 8 sequential test phases run automatically:

```
Phase 1: Target Validation     вҶ’ Connectivity & server info
Phase 2: Tech Fingerprinting   вҶ’ Frameworks, analytics, CMS
Phase 3: Website Crawl         вҶ’ Discover pages & forms
Phase 4: Path Discovery        вҶ’ Hidden directories & sensitive files
Phase 5: Misconfiguration CheckвҶ’ Debug mode, directory listing
Phase 6: SQL Injection Tests   вҶ’ Error-based detection
Phase 7: XSS Tests             вҶ’ Reflected XSS via forms
Phase 8: CSRF / Path Traversal вҶ’ Token checks, file reads
```

#### рҹ§© Chrome Extension Analyzer

1. Navigate to **EXTENSION ANALYZER** tab
2. Paste an Extension ID (32-char string) or full Chrome Web Store URL
3. Use quick-load buttons for popular extensions (uBlock, LastPass, Grammarly)
4. Click **рҹ”Қ ANALYZE**

The analyzer will:
- Fetch metadata from Chrome Web Store
- Download the `.crx` file
- Extract and inspect all JavaScript files
- Flag obfuscated code, dangerous APIs, hardcoded secrets

#### рҹ“Ў Network Monitor

> Requires `scapy` and typically elevated privileges (root/admin)

1. Navigate to **NETWORK MONITOR** tab
2. Set your network interface (`eth0`, `Wi-Fi`, etc.)
3. Click **в–¶ START MONITOR**
4. Live stats update in real time: Packets В· HTTP requests В· Keys found

#### рҹ”Қ Deep Web Crawler

1. Navigate to **DEEP CRAWLER** tab
2. Set target URL, crawl depth (1вҖ“5), and max pages
3. Click **в–¶ START CRAWL**

Crawls the site recursively, then scans all discovered JavaScript files for exposed secrets.
